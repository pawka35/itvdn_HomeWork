Обзор, цель и назначение урока
Клиентский JavaScript предназначенный для того, чтобы превращать статические HTML-документы в интерактивные веб-приложения. Работа с содержимым страницы – это одно из главных назначений JavaScript и цель данного урока показать как это можно делать.
Изучив материал данного занятия, учащийся сможет:
  • Использовать ненавязчивый JavaScript
  • Получать элементы со страницы
  • Работать с DOM, модифицировать и динамически создавать элементы
Содержание урока
  1. Способы подключения JS сценариев
  2. Способы получения данных со страницы
  3. Ненавязчивый JavaScript
  4. Модули и пространства имен
